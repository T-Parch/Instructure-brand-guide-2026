# Style Guide
**Source page:** /written-brand-guide/style-guide
**Last verified:** 2026-05-07
**Version:** v1.0.0

Reference standard: **Merriam-Webster** (NORAM). See 05-glossary.md for Instructure-specific and regional spellings.

---

## Capitalization Rules

| Context | Rule |
|---------|------|
| Headlines, headers, subject lines, buttons, CTAs, bullets | **Sentence case** |
| Published work titles (articles, reports, webinars, videos) | **Title case** |
| All-caps | Design only (eyebrows, labels) — never in body copy |
| Proprietary names | Always capitalize: Instructure, Canvas, Mastery, Parchment, The Instructure Learning Strategy Team |
| Generic features | Always lowercase: edtech, learning management system, artificial intelligence |
| Job titles before a name or in headshots/quotes | Capitalize |
| Job titles after "the" or a person's name | Lowercase |

## Punctuation Quick Reference

| Mark | Rule |
|------|------|
| **Oxford comma** | Always in NORAM ("dogs, RuPaul, and Batman") |
| **Em dash** | No spaces — like this |
| **En dash** | Ranges: K–12, October 5–10 |
| **Hyphen** | Compound modifiers before nouns |
| **Ampersand** | Avoid (design exceptions only) |
| **Exclamation points** | Sparingly — never as a reflexive friendliness signifier |
| **Quotation marks** | Double-tick; punctuation inside; single-tick for nested quotes |
| **Colons** | Introduce lists/explanations; don't capitalize after unless a complete sentence follows |
| **Semicolons** | Join related complete sentences or use as super-commas in complex lists |
| **Parentheses** | Period outside unless entire sentence is parenthetical |

## Dates

| Region | Format | Example |
|--------|--------|---------|
| NORAM | Month DD, YYYY (no ordinals) | March 17, 2026 |
| NORAM (space-limited) | Abbreviate months >5 letters | Feb 3, 2026 |
| APAC | DD Month YYYY, no comma | 5 January 2026 |

---

## Times

| Region | Format | Example |
|--------|--------|---------|
| NORAM | lowercase am/pm, comma + timezone | 10 am, ET |
| NORAM (range) | — | 8:30–11:30 pm, MT |
| APAC | 24-hour | 14:30 |
| Global events | Include UTC in parentheses | 10:00 AM ET (2:00 PM UTC) |

---

## Numbers

| Context | Rule |
|---------|------|
| Headlines / headers | Always numerals |
| Body copy, under 10 | Spell out |
| Body copy, sentence-starting | Spell out |
| Body copy, 10+ | Numerals |
| Times, dates, tables, equations, percentages, ranges | Numerals |
| Plurals | Lowercase s, no apostrophe (10s, 1980s) |

---

## Acronyms
- Spell out on first use (except ATM, WIFI, VIP, CEO)
- Pluralize with lowercase s: MOOCs, APIs, LMSs

---

## Headers
- H1: One per page (page title/headline)
- H2: Section headings
- H3: Subsections — use sparingly
- Never skip levels
- Always sentence case

---

## Links
Hyperlinked text must describe the destination. Don't link trailing punctuation on inline CTAs.

---

## Text Styling
- **Bold:** Emphasize list lead-ins and key phrases — sparingly
- *Italic:* Book, journal, magazine, album titles
- "Quotation marks": Article, podcast, and song titles
